# Data

The original notebook used the California housing dataset from this URL:

`https://raw.githubusercontent.com/ageron/handson-ml/master/datasets/housing/housing.csv`

This project includes `housing_sample.csv` only as a small fallback/sample file because the full dataset could not be downloaded in this environment.

When you run `python src/train.py` on your laptop with internet, the script will automatically download the full dataset and save it here as `housing.csv`.
